## 使用方法：

1.装python，cmd输入

```
pip install pyautogui
pip install keyboard
```

2.打开打开腾讯会议——应用——签到

3.打开TencentMeeting_Sign.Bypass.py

4.把鼠标移动到"点击签到"按钮位置，5秒后会获取坐标

5.泡杯茶，找一张床，躺上去(doge)

注：按空格键停止摸鱼
